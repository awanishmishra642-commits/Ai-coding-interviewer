# 📌 Overview

Nerve IDE is an AI-powered coding interview platform designed to simulate real technical interviews.

Candidates can choose DSA topics, select difficulty levels, solve coding problems inside a Monaco-powered editor, interact with an AI interviewer, and receive instant feedback and performance analysis.

✅ AI-generated interview questions

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/awanishmishra642-commits/ai-coding-interviewer.git
```

Move inside project

```bash
cd ai-coding-interviewer
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file.

```
DATABASE_URL=

SESSION_SECRET=

GROQ_API_KEY=
```
